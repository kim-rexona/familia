HTML

A principios de la década de los 90, el físico Tim Berners Lee, investigador de la Organización Europea para la Investigación Nuclear (CERN), realizó una publicación titulada «HTML Tags», conocida como el origen de HTML.Además, unos años después, fundó la W3C, que es uno de los principales agentes en la historia de HTML.Esta publicación respondía a la necesidad de crear un sistema de hipertexto que les permitiera a los científicos e investigadores del CERN intercambiar información de forma rápida y fácil de referenciar, también tendríamos que tener en cuenta que la investigación nuclear reúne a científicos de diversas disciplinas, por lo que optimizar el intercambio de información resultaba urgente.Es decir,HTML surgió con el objetivo de agilizarles sustancialmente el trabajo y, por supuesto, lo logró.El desarrollo de HTML y otras herramientas y lenguajes web está estrechamente relacionado con la World Wide Web (W3C).El W3C establece principalmente estándares y recomendaciones para la funcionalidad y el crecimiento de la web, y la historia. de HTML y otras herramientas de desarrollo web está estrechamente relacionado con esta organización.En 2004, se creó el Grupo de Trabajo sobre Tecnología de Aplicaciones de Hipertexto Web (WHATWG) en respuesta a la frustración de los fabricantes web con las actualizaciones excesivamente personalizadas. El WHATWG tenía como objetivo abordar la falta de interés del W3C en continuar con las actualizaciones de HTML, que han estado en curso desde 1999, y la última versión, HTML 5, se lanzó en 2008.

Antes de la aparición de HTML, existían dos sistemas de vinculación de documentos electrónicos: los hipervínculos o enlaces y el SGML, un lenguaje de etiquetas. En 1989, un investigador del CERN (la Organización Europea para la Investigación Nuclear), Tim Berners-Lee, asegura que está trabajando en un sistema nuevo de hipertexto para compartir documentos.Cuando su proyecto estuvo listo, lo presentó a una convocatoria para el desarrollo de un sistema de hipertexto para internet. Después, se alió con Robert Cailliau, un ingeniero de sistemas, y juntos presentaron la WorldWideWeb (W3C). Esta fue la propuesta ganadora de la convocatoria.El desarrollo de HTML y otras herramientas y lenguajes web está estrechamente relacionado con la World Wide Web (W3C).A medida que la web creció, también lo hizo HTML. En 1995, se lanzó HTML 2.0, que introdujo características como formularios y tablas, lo que permitió a los desarrolladores crear páginas más interactivas y funcionales. La evolución continuó con HTML 3.2 y HTML 4.0 en los años 90, incorporando mejoras significativas como el uso de hojas de estilo en cascada (CSS), que separaba el contenido de su presentación. Esto permitió un diseño más atractivo y un control más preciso sobre la apariencia de las páginas. Durante esta época, se empezaron a incorporar también elementos multimedia, lo que enriqueció la experiencia del usuario y llevó a un aumento en la complejidad de las aplicaciones webX.La llegada de XHTML en 2000 fue un intento de modernizar HTML bajo las reglas de XML, lo que prometía una estructura más rigurosa y semántica. Sin embargo, la implementación de XHTML enfrentó críticas debido a problemas de compatibilidad con navegadores, lo que llevó a la comunidad a buscar una alternativa más flexible. En 2004, el Consorcio World Wide Web (W3C) comenzó a desarrollar HTML5, un proyecto que se centró en la compatibilidad con los navegadores existentes y en la inclusión de nuevas características como audio, video y API para aplicaciones web. HTML5 fue estandarizado en 2014 y se convirtió en la base para el desarrollo de contenido web moderno, facilitando la creación de aplicaciones más dinámicas e interactivas, y estableciendo un nuevo estándar para la web.

Evolución html

Desde su versión inicial, HTML ha experimentado cambios significativos, cada uno abordando limitaciones anteriores y respondiendo a las necesidades emergentes del entorno digital.la primera versión del navegador web, tiene sus orígenes en 1991, siendo la primera versión html 1.0.En 1995 se publicó la segunda versión, HTML 2.0, que fue una propuesta de estandarización.Dos años después, en 1997, se lanzaron dos nuevas versiones: HTML 3.2 y HTML 4.0.Estos cambios fueron rápidos y los desarrolladores web se adaptaban constantemente a cada modificación.En 1999, HTML 4.01 marcó otro hito en la evolución de HTML ya que este que fue lanzado en 1999, introdujo una separación más clara entre el contenido y la presentación, fomentando el uso de CSS para definir estilos. Esta separación no solo facilitó el trabajo de los desarrolladores, sino que también promovió la accesibilidad, permitiendo a diferentes dispositivos y navegadores mostrar contenido de manera uniforme..HTML 5 se lanzó en 2008 y ahora es la base de todas las versiones actuales. Desde 2014, HTML 5 fue un punto de inflexión en esta evolución. HTML5 modernizó el lenguaje al incorporar nuevas etiquetas semánticas que ayudaron a estructurar mejor el contenido y mejorar su accesibilidad. La inclusión de elementos multimedia, como audio y video, sin necesidad de complementos adicionales, revolucionó la forma en que se presentaban las experiencias en línea. Además, HTML5 introdujo API para crear aplicaciones web ricas, lo que permitió a los desarrolladores construir experiencias interactivas y dinámicas que antes requerían tecnologías más complejas. Este enfoque también fomentó el desarrollo de aplicaciones web progresivas (PWA), que ofrecen una experiencia similar a la de las aplicaciones nativas en dispositivos móviles,es el estándar y aunque continúan nuevas modificaciones y actualizaciones, los cambios son mínimos y fáciles de replicar.Una de las últimas recomendaciones es HTML 5.3, que fue lanzado en 2021, pero no representa un cambio significativo
